03 // Deployment

Deploy instantly with MoRAD.

Complex multimodal inference is hard to ship. The distilled version of MoRA maps raw longitude and latitude directly into the shared embedding space, making the foundation much easier to use in production settings.

  • Privacy Preserving

    No need to process raw mobility or demographic data at inference time.

  • Fast Inference

    A lightweight surrogate network handles coordinate-to-embedding translation.

mora_quickstart.py
import mora

# Load the distilled inference model
model = mora.load_model('morad-128d-national')

# Coordinate-in, representation-out
lat, lon = 31.2304, 121.4737
embedding = model.encode(lat, lon)

print(embedding.shape)
> (128,)

print(embedding[:5])
